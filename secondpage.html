А теперь поговорим о том, как работают формы в html.
То есть как добавлять свои записи, например.
Для этого у нас есть тэг [form]
Чтобы была возможность что-то вводить, используем тэг [input]
У него есть специальный параметр [type] например там может быть text
Если хотим назвать это поле как-нибудь, то используем тэг [label]

Пожалуй, буду теперь строчки подлиннее делать, а то постоянно приходится переключаться на новую и еще и еще и еще - довольно раздражает
Итак, если для [label] указать параметр [for] равный параметру [id] в тэге [input], то при нажатии на название поля нас сразу
перебросит в поле, которое нужно заполнить. У инпутов много параметров. Например, [placeholder] - подсказка в поле ввода.
Есть разные типы инпутов. Из интересного:

  • number - с инструментом для увеличения/уменьшения значения вводимого числа
  • password - чтоб то, что вводит пользователь отображалось звездочками
  • color для выбора цвета
  • date для выбора даты
  • checkbox чтоб ставить галочку (например в тесте с выбором ответа или опросе)
  • radio - тоже что-то подобное
  • submit - кнопка, например, для отправки или перехода куда-то

Еще есть отдельный тэг внутри form, чтобы создать поле для ввода большого объема текста - [textarea]
Можно использовать его, например, если хочешь дать пользователям возможность написать письмо или объемно ответить на вопрос
Кстати, параметр [action] должен указывать скрипт/путь, по которому будет обрабатываться форма (php, java script, удаленный сервер и т д)
Если поставить слэш, то он будет просто обновлять страничку
